# front-end-project 前后端一体项目

## 安装

```sh
git clone https://github.com/myestorm/front-end-project.git
cd front-end-project
npm install -d
```

## 开发

```sh
npm run dev

# server - http://localhost:6001/
# client - http://localhost:3000/ 
```

## 编译

```sh
npm run build
```

## 生产环境下启动

```sh
# 推荐使用PM2
npm start

# 如果只是看看可以使用
npm run server-preview
```
